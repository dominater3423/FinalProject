Introduction

I think most people know what HTML is so I'm not going to bore you with the details. This is not platform specific, all you need is a web browser and a text editor of your choice. Please note though, I will not be covering CSS.

Basic Syntax

In contras to the latex tutorial I will introduce you to the basic sentence, then later shows how it goes all together. Fundamental to Latex is the header tags, the range from one to six, with one being the largest and six smallest. In HTML you always have to close your tags, this won't result in an error, most likely something will be off on your page and you have to go hunting for the problem. That means that HTML does NOT report errors for you, it will run whether or not your code is even correct in the first place. Be careful.

-> <h1>Header One</h1>
->
-> <h2>Header Two</h2>
->
-> <h3>Header Three</h3>
->
-> <h4>Header Four</h4>
->
-> <h5>Header Five</h5>
->
-> <h6>Header Six</h6>

Then the next two will be ordered and unordered list. The syntax for unordered is <ul> and for ordered <ol>.

-> </ol>

Structure

Putting it all together HTML is divided into different sections including a head, body, and footer. You can also add it div tags, but it's generally not recommended to use too many and instead use HTML's native tags. At a bare minimum we will include the charset, title, and a link to your stylesheet in the head. The body contains all the content on your site, then the footer is what you normally see at the bottom of websites with their contact info and license under which the site is under. Below is a basic hello world, and then I'll show a more complex example down below.

-> <!DOCTYPe html>
-> <html>
-> <head>
-> <meta charset="UTF-8">
-> <title>Hello World</title>
-> <link rel="stylesheet" type="text/css" href="style.css">
-> </head>
-> <body>
-> <h1>Hello World</h1>
-> </body>
-> <footer>
-> <p>By: Dominic Carducci</p>
-> </footer>
-> </html>
